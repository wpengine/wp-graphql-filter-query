# wp-graphql-filter-query

Adds taxonomy filtering and aggregation support to WP GraphQL

## Contributing

### Requirements

-   Docker

### Getting Started

To get started with the dev environment run

```
make
```

### Testing

```
make test
```

### Linting

```
make lint
```
